## 다루는 Apple Accessibility 기능

**Vision** — VoiceOver, Zoom, Magnifier, Magnifier for Mac, Display & Text Size, Dynamic Type, Hover Typing, Spoken Content, Audio Descriptions, Accessibility Reader, Braille Access

**Hearing** — Live Captions, Closed Captions, Made for iPhone Hearing Devices, Live Listen, Music Haptics, Sound Recognition, Name Recognition, RTT, Background Sounds, Mono Audio

**Mobility** — Switch Control, Voice Control, AssistiveTouch, Eye Tracking, Head Tracking, Full Keyboard Access, Vocal Shortcuts, Back Tap, Touch Accommodations

**Speech** — Personal Voice, Live Speech, Listen for Atypical Speech

**Cognitive** — Assistive Access, Reduce Motion, Cross-Fade Transitions, Accessibility Reader, Guided Access
